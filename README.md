# Jason McDaniel Portfolio (SIP311)

This repository hosts a simple GitHub Pages site for my UAT work, portfolio artifacts, and my SIP project.

## Publish it with GitHub Pages
1. Open the repo on GitHub.
2. Go to **Settings** → **Pages**.
3. Under **Build and deployment**, set:
   - **Source**: Deploy from a branch
   - **Branch**: `main`
   - **Folder**: `/ (root)`
4. Save.

GitHub will give you the live URL (usually):
`https://spyderdreams.github.io/Jason_McDaniel_portfolio/`

## Site structure
- `index.html` – Home
- `sip.html` – SIP project overview
- `portfolio.html` – Portfolio highlights
- `objectives.html` – Program objectives
- `about.html` – About + contact
- `assets/` – CSS, small scripts, and SVG diagram

---

> Note: This site is intentionally lightweight (plain HTML/CSS) so it loads fast and is easy to maintain.
